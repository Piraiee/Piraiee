# Launch Checklist

1. Confirm content and images.
2. Create the Google Sheet and Apps Script backend.
3. Set `GOOGLE_SCRIPT_URL` in Cloudflare Pages.
4. Add Turnstile if spam protection is needed.
5. Connect `Piraiee/Piraiee` to Cloudflare Pages.
6. Deploy preview.
7. Test forms on the preview URL.
8. Add `piraiee.com` and `www.piraiee.com` custom domains.
9. Copy all DNS records from Squarespace to Cloudflare.
10. Change nameservers at Squarespace only after DNS is mirrored.
